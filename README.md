# angularWeather

1. База данных
Лежала под mysql сервером
Настройки    
    host: 'localhost',
    user: 'root',
    port: 3306,
    password: 'admin',
    database: 'weatherdb'
2. Дамп базы данных лежит в папке(db)
3. Запустить npm install
4. Запустить приложение node server.js -> в браузере http://localhost:3000
